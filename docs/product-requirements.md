# Product Requirements Document (PRD)

## 1. Product Vision
An immersive, highly interactive portfolio application that replicates a macOS desktop experience using modern web technologies. This serves as a creative showcase for a web developer's skills, ensuring a memorable experience for visitors.

## 2. Target Audience
- **Recruiters & Talent Acquisition**: Looking for technical competence and strong presentation.
- **Tech Companies**: Assessing front-end architecture and animation skills.
- **Creative Developers**: Seeking inspiration and engaging web experiences.

## 3. Problem Statement & Value Proposition
Traditional web portfolios are often static and fail to truly engage visitors. This application solves that by introducing an OS-like desktop UI, turning the portfolio from a mere "document" into an "experience." 

## 4. Business Goals & Success Metrics
- **Engagement Time**: Increase average session duration significantly compared to static portfolios.
- **Visual Appeal & Fluidity**: Reach constant 60fps across macOS animations.
- **Conversion**: Increase outreach from recruiters or clients.

## 5. Core Features
- **MacOS Desktop Interface**: Top Menu Bar, Glassmorphism Dock, and Desktop File Grid.
- **Window Management**: Draggable, closable, minimize/maximize capable windows with proper z-index focus.
- **Application Views**: 
  - *About Me* (Finder UI)
  - *Projects* (Photos Grid UI)
  - *Resume* (PDF Viewer UI)
- **Fluid Animations**: Smooth component mounting, window scaling, and dock magnification using Framer Motion and GSAP.
